# 🏦 Sistema Bancário em C#

Sistema bancário desenvolvido em C# utilizando Programação Orientada a Objetos (POO), com foco na organização do código, criação de classes e manipulação de objetos.

O projeto simula operações básicas de uma conta bancária em um ambiente de console, servindo como prática dos principais conceitos de orientação a objetos.

## 🚀 Funcionalidades

- Cadastro de cliente
- Criação de conta bancária
- Depósito
- Saque
- Consulta de saldo
- Exibição dos dados da conta
- Menu interativo
- Tratamento de entradas inválidas

## 📚 Conceitos praticados

- Programação Orientada a Objetos (POO)
- Classes e Objetos
- Propriedades (`get` e `set`)
- Construtores
- Encapsulamento
- Relacionamento entre classes
- Métodos
- Estruturas condicionais (`switch`)
- Laços de repetição (`do...while`)
- Tratamento de exceções (`try/catch`)
- Utilização do tipo `decimal` para valores monetários

## 🛠️ Tecnologias

## 🎯 Objetivo

Aplicar os conceitos fundamentais da Programação Orientada a Objetos por meio de um projeto prático, reforçando boas práticas de organização, reutilização de código e preparação para o desenvolvimento de aplicações maiores utilizando ASP.NET Core.
